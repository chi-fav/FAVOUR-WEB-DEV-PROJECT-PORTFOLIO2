HOW TO CREATE AN HTML FILE

  1. Install the VS code into your system if not already installed
  2. Launch the vs code
  3. click on file drop down menu located at the left side of the title bar in the vs code
  4. Click on new file, the second option on the drop down menu
  5. Type in you desired name for your file making sure to add the html extension .html e.g DesiredName.html or DesiredName.htm
  6. Click on create file, the first option on lower right of the dialogue box
  7. You just fiished creating an html file
